# uso:
./c-calculo <número> <operação> <número>  
operações matemáticas disponíveis: +, -, *, /, ^,r (raiz).  
caso multiplicação não funcione, utilize asterisco entre aspas  
 
exemplos:  
$./c-calculo 64 + 2    
64 + 2 = 66  
  
$./c-calculo 64 '*' 2  
64 * 2 = 128  

$./c-calculo 64 r 2  
64 índice 2 = 8
